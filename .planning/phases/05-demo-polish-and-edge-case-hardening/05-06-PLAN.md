# P5-06 Plan - Finalize TRA Lock Semantics And Same-Screen Cockpit Observation

## Objective

Correct the interactive TRA lock semantics so the browser only unlocks deep reverse travel after the current snapshot actually reaches the `-14°` lock point with `L4` satisfied, and rework the desktop cockpit layout so VDT adjustment no longer pulls the logic board out of view.

## Context

- `P5-05` introduced an `L4`-gated deep-travel lock and a denser cockpit layout, but user feedback exposed two remaining issues.
- The browser could still pre-arm the deeper reverse range before the current snapshot had actually reached the `-14°` lock point.
- The left-side cockpit remained too tall, and the VDT control still sat low enough that watching the right-side logic board during adjustment felt awkward.
- The project constitution still requires the lock behavior to stay on the existing `POST /api/lever-snapshot` evidence plane without changing `controller.py` truth.

## Tasks

1. Tighten the TRA lock contract so the UI stays in the `-14°..0°` range until the current snapshot itself reaches the `-14°` lock point with `L4` active, while the backend still blocks direct deep-range requests when the gate is not ready.
2. Move the VDT controls into the top lever console, compact the remaining condition panel, and make the desktop logic board remain visible while the left column scrolls.
3. Extend regression coverage and smoke evidence so the corrected lock semantics and same-screen layout flow stay protected by automated validation.

## Success Criteria

- When `L4` is not currently satisfied, the browser slider can only drag inside `-14°..0°`; it does not unlock the deeper reverse range early.
- Once the current snapshot reaches `-14°` with `L4` active, the deeper `-14°..-32°` range unlocks.
- Desktop use keeps the VDT controls near the top of the cockpit and keeps the right-side logic board visible during adjustment.
- Shared validation and demo smoke both pass with the corrected lock semantics included in the automated evidence plane.

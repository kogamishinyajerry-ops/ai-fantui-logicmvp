# P5-07 Plan - Clarify Conditional TRA Deep-Range Drag And Relax Cockpit Density

## Objective

Correct the demo cockpit so the TRA slider always shows the full `-32° .. 0°` visual range while only opening the `-32° .. -14°` deep-reverse band after the `L4` boundary logic is satisfied, and rebalance the desktop layout so the lever deck feels spacious instead of cramped.

## Context

- `P5-06` moved VDT controls higher and made the lock state more visible, but user feedback exposed two remaining mismatches.
- The intended interaction model is not “reach `-14°` first, then unlock”; it is “keep the full visual slider range, but only allow free dragging inside `-14° .. 0°` until the `L4` boundary condition is ready.”
- The current browser still lets the thumb drift into the deep band before the backend snaps it back, which feels wrong even though the API payload is safe.
- The left-side cockpit is still visually crowded: preset cards, helper copy, condition controls, and VDT controls compete for the same density budget.

## Tasks

1. Preserve the full `-32° .. 0°` visual slider range while clamping browser-side free dragging to `-14° .. 0°` until the backend payload reports that the `L4` boundary unlock is ready.
2. Rebalance the desktop cockpit layout so the lever deck, presets, and condition area breathe more clearly without hiding the right-side logic board during VDT adjustment.
3. Extend automated coverage so the clarified drag-band semantics and relaxed desktop density stay protected by unit/static checks and the existing demo smoke suite.

## Success Criteria

- The TRA slider still renders `-32° .. 0°`, but when `L4` is not ready the browser only allows free dragging inside `-14° .. 0°`.
- Once the `L4` boundary unlock is ready, the browser allows free dragging across the full `-32° .. 0°` range without changing the existing API contract.
- The desktop cockpit presents larger spacing and less control/button crowding while keeping VDT adjustment and the right-side logic board easy to read together.
- Shared validation and demo smoke both pass with the clarified drag-band behavior included in the automated evidence plane.

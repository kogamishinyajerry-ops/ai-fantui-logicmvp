## Q&A 预案

| 观众问题 | 推荐回答 |
|-----------|---------|
| Canvas上蓝色高亮是什么？ | AI讨论过的节点，和Truth Engine的节点状态是两套独立的视觉层 |
| AI会编造故障原因吗？ | 不会。AI只读Truth Engine的输出，Truth Engine是100%确定性的、无概率的 |
| 为什么不直接用大语言模型推理控制逻辑？ | 控制逻辑不能概率推断。8+8=16是对的，但概率上"可能是16"会导致事故 |
| 支持737还是380？ | 支持任何定义了逻辑节点和条件的控制系统。我们有4套的adapter |
| 634个测试都测什么？ | 单元测试、对抗测试（注入错误输入）、集成测试（端到端API）、8/8对抗契约 |
| 一次诊断要多长时间？ | 敏感性分析20秒（含20次API调用），单次诊断<1秒 |
| 为什么不直接在Canvas上改参数？ | Canvas只显示Truth Engine的结果，修改需要通过API。Truth Engine是唯一的事实来源 |

架构需求 · Architecture AR-01 → AR-03 · 共 3 条

AR-01 电气总线双路供电 115VAC 540VDC

系统区分 115VAC 单相（TLS / PLS 解锁）与 540VDC 三相（TRCU / PDU 驱动），两路总线独立控制，不得混用。

来源: C919 电气架构规范 §4.2
AR-02 EICU 双通道冗余 CMD1 CMD2

EICU 两个控制通道独立评估条件，任一通道健康且条件满足即发命令；双通道同时失效进入 SF 故障状态。

DO-178C DAL-A · LGCU1_WOW / LGCU2_WOW 双通道
AR-03 FADEC 命令接口 FADEC

控制逻辑不直接驱动 PDU，而是发出 FADEC_Deploy_Command / FADEC_Stow_Command，由 FADEC 执行并回送确认。

状态转移需求 · State transitions SM-01 → SM-07 · 全局故障 SM-SF

SM-01 S0 → S1 地面装备 S1

Selected_MLG_WOW = True ∧ engine_running = True，进入"地面装备"等待飞行员拉杆。

SM-02 S1 → S2 单相解锁供电 S2

SW1 激活 (TRA ∈ -6.2°~-1.4°) ∧ LGCU WOW ∧ engine_running → SinglePhaseUnlockPower_On。

SM-03 S2 → S3 三相作动供电 S3

TLS_Unlocked ∧ PLS_L_Unlocked ∧ SW2 激活 (TRA ≤ -5.0°) → ThreePhaseTRCUPower_On (540VDC)。

SM-04 S3 → S4 展开中 S4

FADEC_Deploy_Command 已发 ∧ Unlock_Confirmed → FADEC 驱动 PDU，TR_Position 上升。

SM-05 S4 → S5/S6 反推出力 S5 S6

TR_Position ≥ 90% (VDT90) → S5 慢车；继续拉杆 TRA ≤ -28° → S6 最大反推。

SM-06 S6/S5 → S7 减速待收 S7

TRA 回到 0° (推杆) → 发出 FADEC_Stow_Command，进入收起准备态。

SM-07 S7 → S8/S9/S10 → S0 S8 S9 S10

S8 收起中 → S9 TR_Position ≤ 0% ∧ TLS/PLS 重锁 → S10 断电 → S0 等待下次接地。

SM-SF 任意状态 → SF 故障中止 SF

EICU 双通道失效 / 传感器全失联 / 位置异常 / 飞行员主动中止 → 切断所有命令，需人工重置。

最高优先级，全局转移目标

功能需求 · Functional requirements FR-C1 → FR-C3 · 共 3 条

FR-C1 到位确认才允许最大推力 S5→S6

与 FANTUI SR-04 等价：TR_Position < 90% 时不进入 S6；TRA 深拉区 (≤ -28°) 在 S5 确认前不可用。

FR-C2 锁机构到位才允许断电 S10

进入 S10 的条件之一是 TR_Stowed_And_Locked = True，确认翻转斗已物理归位且所有锁重新锁定。

FR-C3 PDU 过温保护 S4-6→S7

PDU 电机过温信号 (overtemp) 在 S4/S5/S6 立即发出 FADEC_Stow_Command 并跳转 S7。

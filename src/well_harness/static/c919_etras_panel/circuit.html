WOW FILTER — 落地确认逻辑

对 LGCU1 OR LGCU2 的 WOW 信号做 0.5s SET 延时（防弹跳）、0.0s RESET（即时响应离地）。输出 tr_wow 驱动状态机从 S0→S1。

CMD2 — 解锁电源时序

TRA 进入反推角度后，先接通单相解锁电源（1-ph unlock），计时 2.0s 后切换为三相 TRCU 全功率。防止冷状态瞬间大电流冲击。

CMD3 SR LATCH — 展开使能

  • SET：反推不在收起+锁定位置 && 无过温故障
  • RESET：tr_stowed_and_locked = 1（需 0.5s 驻留）
  • Latch 状态 cmd3_latch 阻止意外展开命令

故障安全路径

  • ETRAS_OVER_TEMP → 立即 STOW CMD，CMD3 SET 禁止
  • TR_INHIBITED → 禁止任何展开命令
  • 空中离地（!WOW）→ 立即 STOW，状态回 S0
  • N1K ≥ deploy limit → 阻止展开

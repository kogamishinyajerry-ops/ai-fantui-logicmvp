L1 — TLS 解锁起始

  • reverser_not_deployed_eec — EEC 未报告已展开

输出 tls_115vac_cmd（115 VAC 液压锁解锁电路上电）。

L2 — ETRAC 上电

  • engine_running == True
  • reverser_inhibited == False
  • SW2 == True — TRA -5.0°/-9.8° 区间机械开关
  • eec_enable == True

输出 etrac_540vdc_cmd（540 VDC 电机控制器主功率）。

L3 — 展开执行授权（三路扇出）

  • engine_running == True
  • reverser_inhibited == False
  • tls_unlocked_ls — TLS 位置反馈已解锁
  • n1k < max_n1k_deploy_limit — 风扇转速限制
  • tra_deg ≤ logic3_tra_deg_threshold（典型 -11.74°）

同时输出 eec_deploy_cmd、pls_power_cmd、pdu_motor_cmd。

L4 — 最终油门解锁

  • deploy_90_percent_vdt — VDT 位置传感器 ≥ 90%
  • reverse_travel_min < tra_deg < reverse_travel_max（典型 −32° ~ 0°）
  • engine_running == True

输出 throttle_electronic_lock_release_cmd，开放反推深拉区。
